Required for self-hosting:

- structs
- enums
- fixed arrays
- pointers
- address-of / dereference
- function calls
- if / while / for / switch
- integer and bool operations
- casts
- compile-time constants
- fixed byte-array string literals
- basic file I/O through runtime calls
- arena allocation
- diagnostics
- multi-file compilation or temporary include mechanism

Explicitly not required yet:

- first-class string
- generics
- modules/packages
- closures
- interfaces/traits
- operator overloading
- advanced compile-time execution
